let obj = {
   firstName: "Jay",
   lastName: "Elie",
   occupation: "Instructor"
}

obj.firstName;
obj.lastName;
obj["occupation"];

obj.hobby = "Tennis";
delete obj.occupation;

You use the bracket notation when you need to evaluate some expression or pass in a variable to get the name of the key, but when you know the name of the key and it is not a variable or expression, always use the dot notation.
